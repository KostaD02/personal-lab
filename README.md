# Personal Lab

A documented journey of building a clean, reproducible self-hosted environment.

## System Architecture

- **Host:** Arch Linux
- **Storage:** Hybrid SSD (System/DBs) + HDD (Media/Backups)

## Project Structure

- `/docs`: Step-by-step guides (Arch install, Docker setup).
- `/stacks`: Docker Compose files organized by category.
- `/scripts`: Maintenance and backup automation.
